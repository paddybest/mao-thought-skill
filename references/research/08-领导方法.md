# 领导方法

## 核心命题

**干部决定一切** —— 正确路线确定后，干部就是决定的因素

---

## 一、干部的识别与选拔

### 1.1 识别干部的标准

毛泽东在六届六中全会(1938)上提出系统的干部识别方法：

**四条根本标准：**

| 标准 | 内涵 | 考察方法 |
|------|------|----------|
| 执行路线 | 坚定执行党的路线 | 长期观察 |
| 遵守纪律 | 严格遵守党的纪律 | 关键时刻 |
| 联系群众 | 密切联系人民群众 | 日常表现 |
| 独立判断 | 独立开展工作的能力 | 复杂局面 |

**核心原则：**
> 判断一个人，必须看他 **一生的全部历史**，而不是 **一时一事**。

### 1.2 选拔干部的原则

**德才兼备，以德为先：**

```
德 ━━━━━━━━━━━━━━━━━━ 才
  │                    │
  │ 政治方向正确        │ 工作能力突出
  │ 群众关系密切        │ 能独立解决问题
  │ 个人品质高尚        │ 主动积极工作
  │                     │
  └───── 二者缺一不可 ──┘
```

**反对两种倾向：**
- 单纯强调能力，忽视政治品质
- 单纯强调政治，忽视实际能力

## 二、干部的使用艺术

### 2.1 领导的核心职责

**两项基本职责：**

1. **出主意** - 制定正确的路线方针政策
2. **用干部** - 选拔、使用、培养干部

> "领导者的责任，归结起来，主要地是出主意、用干部两件事。"

### 2.2 用人所长

**基本原则：**
- 金无足赤，人无完人
- 用其所长，避其所短
- 看主流，看大节

**典型论述：**
> 骄傲使人落后，虚心使人进步。
> 要善于团结同志，包括那些和自己意见不一致的人。

### 2.3 容人之量

**包容的境界：**

| 层次 | 表现 |
|------|------|
| 初级 | 容人之短 |
| 中级 | 用人之长 |
| 高级 | 化敌为友 |
| 极致 | 团结一切可以团结的人 |

## 三、干部的培养教育

### 3.1 培养干部的五个途径

**《关心群众生活，注意工作方法》(1934)中的实践经验：**

| 途径 | 方法 | 目的 |
|------|------|------|
| 指导工作 | 给任务、定目标 | 锻炼能力 |
| 提高水平 | 组织学习、培训 | 提升素质 |
| 检查工作 | 总结经验、纠正错误 | 改进工作 |
| 说服教育 | 耐心细致的思想工作 | 统一认识 |
| 帮助困难 | 解决工作生活中的问题 | 关怀爱护 |

### 3.2 指导的艺术

**正确指导的要点：**

1. **给以方向** - 明确工作目标
2. **给以方法** - 传授工作方法
3. **给以条件** - 提供必要支持
4. **给以责任** - 明确责任担当
5. **给以信任** - 放手让他们工作

> "要让他们放手工作，又要给他们及时指导。"

### 3.3 提高干部水平

**培养方式：**

```
理论学习 ━━━━━━━━━━━━━━━━━━ 实践锻炼
   │                            │
   │ 学习马克思主义基本原理      │ 在实际工作中增长才干
   │ 学习党的路线方针政策        │ 处理复杂问题的能力
   │ 学习历史文化知识            │ 独立判断的本领
   │                            │
   └──────── 理论联系实际 ───────┘
```

## 四、干部的检查与监督

### 4.1 检查工作的方法

**检查的三个层次：**

1. **事前指导** - 工作开始前的方向指引
2. **事中监督** - 工作进行中的及时纠正
3. **事后总结** - 工作完成后的经验教训

**正确做法：**
> 既要交任务，又要交方法；既要检查结果，又要帮助过程。

### 4.2 总结经验的方法

**总结的四个步骤：**

| 步骤 | 内容 | 要求 |
|------|------|------|
| 回顾 | 全面回顾工作过程 | 客观真实 |
| 分析 | 找出成功与失败原因 | 深入透彻 |
| 归纳 | 提炼规律性认识 | 上升到理论 |
| 运用 | 指导下一步工作 | 理论联系实际 |

### 4.3 纠正错误的艺术

**对待犯错误的干部：**

| 错误性质 | 处理方式 |
|----------|----------|
| 轻微错误 | 说服教育，耐心帮助 |
| 严重错误 | 开展批评，限期改正 |
| 屡教不改 | 开展斗争，严肃处理 |

**重要原则：**
> 对犯错误的同志，要采取 **治病救人** 的方针。

## 五、关心爱护干部

### 5.1 五个关心方面

| 方面 | 内容 | 意义 |
|------|------|------|
| 工作指导 | 及时正确的领导 | 帮助成长 |
| 学习提高 | 组织学习培训 | 提升素质 |
| 检查帮助 | 总结经验纠正错误 | 改进工作 |
| 说服教育 | 耐心细致的思想工作 | 统一认识 |
| 帮助困难 | 解决实际困难问题 | 温暖人心 |

### 5.2 避免两种错误倾向

**错误倾向对照：**

| 错误倾向 | 表现 | 危害 |
|----------|------|------|
| 自由主义 | 放任不管 | 错误蔓延 |
| 命令主义 | 包办代替 | 压抑创造 |

**正确做法：**
> 既不放任自流，又不包办代替；既要严格要求，又要热情帮助。

## 六、领导方法的核心原则

### 6.1 从群众中来，到群众中去

**领导方法的基本路线：**

```
群众 ━━━━→ 意见 ━━━━→ 领导 ━━━━→ 办法 ━━━━→ 群众
      调查研究              总结提高          宣传组织
      集中起来              形成政策          坚持下去
```

**"一般和个别相结合"的领导方法：**

1. **从个别到一般** - 从具体案例中总结普遍规律
2. **从一般到个别** - 用普遍规律指导具体工作

### 6.2 解剖麻雀的工作方法

> 要学会 "解剖麻雀"，从个别中看到一般。

**具体做法：**
- 深入基层，深入实际
- 解剖一个典型，了解全面情况
- 总结经验教训，指导全面工作

### 6.3 抓中心环节的艺术

**工作方法论：**

| 原则 | 说明 |
|------|------|
| 中心工作 | 抓住最重要的工作 |
| 带动一般 | 以中心工作带动全局 |
| 统筹兼顾 | 注意工作的系统性 |

## 七、领导作风的培养

### 7.1 三大作风

| 作风 | 内涵 | 要求 |
|------|------|------|
| 理论联系实际 | 把马克思主义基本原理同具体实际相结合 | 学以致用 |
| 密切联系群众 | 一切为了群众，一切依靠群众 | 服务人民 |
| 批评与自我批评 | 勇于揭露问题，善于改正错误 | 永葆先进 |

### 7.2 反对不良作风

| 不良作风 | 表现 | 纠正方法 |
|----------|------|----------|
| 主观主义 | 脱离实际瞎指挥 | 调查研究 |
| 宗派主义 | 拉帮结派搞小圈子 | 顾全大局 |
| 形式主义 | 做表面文章不务实 | 注重实效 |

## 八、关键启示

### 8.1 领导方法的辩证法

```
关心爱护 ━━━━━━━━━━━━━━ 严格要求
    │                      │
    │ 关心是严格要求的基础    │
    │ 严格要求是关心的体现    │
    │                      │
    └──── 辩证统一 ─────────┘
```

### 8.2 实践标准

**检验领导方法正确性的标准：**
1. 是否调动了干部的积极性
2. 是否提高了干部的能力水平
3. 是否解决了实际问题
4. 是否推进了事业发展
