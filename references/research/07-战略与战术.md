# 战略与战术

## 核心命题

**"枪杆子里面出政权"** —— 武装斗争是中国革命的核心形式

---

## 一、战略思维的根本原则

### 1.1 战争是政治斗争的最高形式

毛泽东在《战争和战略问题》(1938)中系统阐述：

> 马克思列宁主义的道理归根到底就是一句话：**"造反有理"**
> 革命的中心任务和最高形式是武装夺取政权，是战争解决问题。

**核心洞察：**
- 在半殖民地半封建的中国，议会道路走不通
- 合法斗争的空间极为有限
- 战争是主要斗争形式，军队是主要组织形式

### 1.2 战略与战术的辩证关系

```
战略上藐视敌人 ━━━━━━━━━━━━━━━━━━ 战术上重视敌人
    (整体、大局)                      (局部、具体)
```

**经典论述：**
> "在战略上我们要藐视一切敌人，在战术上我们要重视一切敌人。"

## 二、游击战争的战略地位

### 2.1 游击战的重要性

《抗日游击战争的战略问题》(1938)系统论述了游击战的战略价值：

**游击战争具有六大战略作用：**

1. **牵制消耗** - 分散敌人兵力，打乱其战略部署
2. **创建根据地** - 在敌后建立稳固的抗日基地
3. **发动群众** - 组织沦陷区人民参与抗日斗争
4. **正规军补充** - 为主力部队提供兵源和物资
5. **心理威慑** - 动摇敌占区民心士气
6. **长期持久** - 以空间换时间，消耗敌人实力

### 2.2 保存自己与消灭敌人

**辩证关系：**
- 游击战争的基本方针是**保存自己、消灭敌人**
- 二者是对立统一的关系
- 在敌强我弱的条件下，游击队必须采取**分散游击**的方式

### 2.3 根据地建设的战略意义

**五种根据地类型：**

| 类型 | 特点 | 战略价值 |
|------|------|----------|
| 山地根据地 | 地形优势 | 天然屏障 |
| 河湖根据地 | 水网纵横 | 机动灵活 |
| 平原根据地 | 人口密集 | 资源丰富 |
| 沿海根据地 | 对外联络 | 国际支持 |
| 地下根据地 | 隐蔽性强 | 长期坚持 |

## 三、持久战的战略思想

### 3.1 三大阶段论

《论持久战》(1938)将抗日战争分为三个阶段：

```
第一阶段：战略防御 ── 敌人进攻，我方防御
    ↓
第二阶段：战略相持 ── 双方拉锯，敌我力量消长
    ↓
第三阶段：战略反攻 ── 我方反攻，收复失地
```

**阶段特征对比：**

| 阶段 | 敌方 | 我方 | 国际形势 |
|------|------|------|----------|
| 防御相持 | 强进攻 | 弱防御 | 观望 |
| 相持 | 减弱 | 成长 | 转变 |
| 反攻 | 劣势 | 优势 | 有利 |

### 3.2 持久战的力量对比变化

**力量消长的规律：**
- 敌人力量**逐步削弱**（战线过长、资源不足、国际反对）
- 我方力量**逐步成长**（团结扩大、经验积累、力量整合）
- 敌方困难**日益增长**，我方优势**日益明显**

## 四、战略指导的艺术

### 4.1 主动性、灵活性、计划性

**主动性** - 掌握战场主动权，不被敌人牵着鼻子走

> 游击队要善于寻找敌人的弱点，主动出击，出其不意。

**灵活性** - 根据情况变化灵活调整战术

| 情况 | 对策 |
|------|------|
| 敌小我大 | 包围聚歼 |
| 敌大我小 | 分散游击 |
| 地形复杂 | 伏击袭击 |
| 地形开阔 | 运动歼敌 |

**计划性** - 没有计划的战争是盲目的战争

> 游击战争要胜利，必须有周密的计划和切实的准备工作。

### 4.2 集中兵力与分散兵力

**辩证法则：**
- 当需要**打歼灭战**时，集中兵力
- 当需要**保存自己**时，分散兵力
- 根据**战略阶段**和**任务性质**灵活转换

## 五、统一战线中的战略思维

### 5.1 又联合又斗争

**统一战线的战略原则：**
- **以斗争求团结则团结存**
- **以退让求团结则团结亡**
- 斗争是手段，团结是目的

### 5.2 战略上把握全局

**毛泽东的战略全局观：**

1. **全国视野** - 把局部问题放在全国大局中考虑
2. **长远眼光** - 看到战争发展的总趋势
3. **各方关系** - 处理好敌、我、友三方关系
4. **主动权** - 牢牢掌握战略主动权

## 六、战略思维的实践运用

### 6.1 从实际出发制定战略

**战略制定的依据：**
- 敌我力量对比
- 国际环境变化
- 国内政治形势
- 地理条件限制

### 6.2 战略服从政治

> 战争是政治的继续，战略必须服从于政治。

**战略决策的政治考量：**
- 是否有利于团结最广大人民
- 是否有利于争取中间势力
- 是否有利于孤立顽固势力

---

## 七、关键启示

### 7.1 战略思维的本质特征

| 特征 | 内涵 |
|------|------|
| 全局性 | 从整体高度把握问题 |
| 长远性 | 看到事物发展的趋势 |
| 辩证性 | 把握对立统一的规律 |
| 灵活性 | 根据情况随机应变 |
| 坚定性 | 战略方向不可动摇 |

### 7.2 战略与战术的统一

```
战略 ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 战术
  │                                    │
  │  战略决定战术的方向                │
  │  战术实现战略的目标                │
  │                                    │
  └────────── 二者相互依存 ─────────────┘
```

### 7.3 实践检验标准

**检验战略正确性的标准：**
1. 是否符合客观实际
2. 是否能够最终胜利
3. 是否得到群众支持
4. 是否有利于力量成长
