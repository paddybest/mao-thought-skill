# 政策与策略

## 核心命题

**政策和策略是党的生命** —— 正确的政策策略是革命胜利的保障

---

## 一、统一战线的策略思想

### 1.1 又联合又斗争的策略

**统一战线的根本策略：**

| 情况 | 策略 |
|------|------|
| 进步势力 | 发展壮大 |
| 中间势力 | 争取团结 |
| 顽固势力 | 又联合又斗争 |

**斗争的原则：**
- **有理** - 有充分的理由
- **有利** - 取得实际的成果
- **有节** - 适可而止

### 1.2 发展进步势力，争取中间势力，孤立顽固势力

**三大势力的划分：**

```
进步势力 ━━━━━━━━━━━━━━━━━━ 中间势力 ━━━━━━━━━━━━━━━━━━ 顽固势力
   │                            │                            │
   工人、农民、革命知识分子      民族资产阶级、开明绅士       大地主、大资产阶级
   共产党领导的革命力量         地方实力派、民主党派          国民党内的投降派
   │                            │                            │
   发展、扩大                   争取、团结                   孤立、反对
```

### 1.3 统一战线中的独立性问题

**独立自主的原则：**
- 在统一战线中保持自己的独立性
- 对国民党采取又团结又斗争的方针
- 既反对 "左" 的关门主义，又反对右的投降主义

## 二、阶级斗争与民族斗争的一致性

### 2.1 民族斗争与阶级斗争的辩证关系

**《统一战线中的独立自主问题》(1938)核心思想：**

| 关系 | 说明 |
|------|------|
| 统一性 | 民族解放是阶级解放的前提 |
| 独立性 | 各阶级保持自己的特殊利益 |
| 调整性 | 在民族斗争中调整阶级关系 |

**具体运用：**
- 阶级利益服从民族利益
- 但不放弃阶级立场
- 在民族斗争中保存和发展革命力量

### 2.2 "一切通过统一战线" 的错误

**批判 "一切通过统一战线" 的口号：**

| 错误表现 | 危害 |
|----------|------|
| 把领导权拱手让人 | 丧失主动权 |
| 放弃独立自主 | 变成尾巴主义 |
| 束缚自己手脚 | 无法开展工作 |

**正确做法：**
> 独立自主，既统一，又独立；既团结，又斗争。

## 三、战略上藐视敌人，战术上重视敌人

### 3.1 辩证的战略观

**核心命题：**

```
战略上 ━━━━━━━━━━━━━━━━━━━━━━ 战术上
   │                              │
   藐视一切敌人                  重视一切敌人
   看到胜利的前途                看到道路的曲折
   树立必胜信心                  采取谨慎态度
   │                              │
   └──────── 辩证统一 ─────────────┘
```

### 3.2 具体应用

**对付敌人的策略原则：**

| 敌人力量 | 战略态度 | 战术方法 |
|----------|----------|----------|
| 强大敌人 | 持久战 | 消耗战 |
| 弱小敌人 | 速决战 | 歼灭战 |
| 孤立敌人 | 各个击破 | 集中兵力 |

## 四、原则的坚定性与策略的灵活性

### 4.1 二者的辩证关系

```
原则性 ━━━━━━━━━━━━━━━━━━━━━━ 灵活性
   │                              │
   坚持正确方向                  适应情况变化
   坚守根本利益                  采取变通办法
   不可动摇                      随机应变
   │                              │
   └──── 原则性是灵活性的前提 ─────┘
```

### 4.2 退让与进攻的辩证法

**《论持久战》中的战略转变：**

| 阶段 | 战略 | 策略 |
|------|------|------|
| 防御阶段 | 战略防御 | 游击战、运动战 |
| 相持阶段 | 战略相持 | 广泛发动群众 |
| 反攻阶段 | 战略反攻 | 全面反攻 |

**"以退为进"的策略艺术：**

> "以退让求团结则团结亡，以斗争求团结则团结存。"

## 五、让步与斗争的策略

### 5.1 积极让步与消极退让

**区分两种让步：**

| 类型 | 性质 | 效果 |
|------|------|------|
| 积极让步 | 策略性让步 | 争取主动 |
| 消极退让 | 投降主义 | 丧失阵地 |

**积极让步的策略：**
- 为了团结抗日作出必要让步
- 让步不影响根本利益
- 让步是为了更好的进攻

### 5.2 让步的限度

**判断让步是否正确的标准：**

| 标准 | 要求 |
|------|------|
| 政治上 | 是否有利于抗日民族统一战线 |
| 力量上 | 是否保存和发展了自己的力量 |
| 时间上 | 是否争取到了喘息的时机 |

## 六、政策制定的方法论

### 6.1 从实际出发的政策制定

**政策制定的依据：**

```
客观实际 ━━━━━━━━━━━━━━━━━━ 政策制定
    │                            │
    敌我力量对比                 制定正确政策
    国内政治形势                 │
    国际环境变化                 贯彻执行
    群众觉悟程度                 │
    │                            实践检验
    └────────────────────────────┘
```

### 6.2 政策的灵活性与原则性

**政策运用的辩证法：**

| 方面 | 要求 |
|------|------|
| 基本政策 | 不可轻易变动 |
| 具体政策 | 可根据情况调整 |
| 策略手段 | 灵活多样 |

**"因时因地制宜"的原则：**

> 政策必须因时因地制宜，不可以千篇一律。

## 七、团结与斗争的策略运用

### 7.1 团结的目的

**团结的出发点和落脚点：**
- 团结是为了壮大革命力量
- 团结一切可以团结的人
- 在团结中发展，在发展中团结

### 7.2 斗争的目的

**斗争的出发点和落脚点：**
- 斗争是为了实现团结
- 斗争是团结的手段
- 以斗争求团结则团结存

### 7.3 团结与斗争的辩证统一

```
团结 ━━━━━━━━━━━━━━━━━━━━━━ 斗争
   │                            │
   目的                          手段
   出发点                        方法
   归宿                          过程
   │                            │
   └──── 团结中有斗争 ───────────┘
   └──── 斗争中有团结 ───────────┘
```

## 八、关键策略原则

### 8.1 "有理有利有节" 的策略原则

| 原则 | 具体要求 | 运用场景 |
|------|----------|----------|
| 有理 | 师出有名 | 斗争要有正当理由 |
| 有利 | 取得实利 | 斗争要能获得实际成果 |
| 有节 | 适可而止 | 斗争要有一定限度 |

### 8.2 政策执行的策略

**政策执行的辩证法：**

| 方法 | 说明 |
|------|------|
| 一般与个别相结合 | 先试点后推广 |
| 领导与群众相结合 | 集中智慧执行 |
| 原则性与灵活性相结合 | 因地制宜执行 |

## 九、统一战线中的独立自主

### 9.1 独立自主的内涵

**独立自主的三个层面：**

| 层面 | 内容 |
|------|------|
| 政治上 | 保持自己的政治路线 |
| 组织上 | 保持自己的组织系统 |
| 军事上 | 保持自己的武装力量 |

### 9.2 独立自主的目的

> 在统一战线中保持独立自主，是为了 **发展进步势力，争取中间势力，孤立顽固势力**，最终战胜敌人。

## 十、关键启示

### 10.1 政策策略的辩证法

**政策策略制定的启示：**

1. 一切从实际出发
2. 具体问题具体分析
3. 原则性与灵活性统一
4. 眼前利益服从长远利益
5. 局部利益服从全局利益

### 10.2 策略检验标准

**正确策略的标准：**
1. 是否有利于发展革命力量
2. 是否有利于团结大多数
3. 是否有利于最终胜利
4. 是否经得起实践检验
